# Investment Tracker App

This is a simple **Investment Tracker App** built with **React** and **Tailwind CSS**. The app allows users to **track their investments**, view **investment metrics**, and access a **financial dictionary** for better understanding of stock-related terms.

---

## 🚀 Features

- 📈 **Add, view, and delete investments**
- 🧮 **Automatically calculated stock metrics** such as P/E, P/B, P/S, and PEG Ratio
- 📚 **Interactive financial dictionary** with search and related term suggestions
- 🔎 **Explanations of each metric** with real-world examples
- 🧠 **Quiz mode** to test your financial knowledge (coming soon)

---

## 🛠️ Technologies Used

- **React**
- **Tailwind CSS**
- **JavaScript**
- **Chart.js / Recharts**
- **Jest**

---

## 📊 Metrics Included

Each investment automatically calculates the following key metrics:

- **P/E Ratio (Price to Earnings)**
- **P/B Ratio (Price to Book)**
- **P/S Ratio (Price to Sales)**
- **PEG Ratio (Price/Earnings to Growth)**

---

## 📘 Financial Glossary

Navigate to the **Glossary** tab to explore over 30 financial terms. Includes:

- Simple definitions
- Real examples
- Related concepts

---

## 🎯 Goals

The goal of this app is to **make investing more understandable** for beginners and to provide a clear, easy-to-use interface to **track and learn** from your portfolio.
